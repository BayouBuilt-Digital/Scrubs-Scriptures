Key-Insight dark gradient

A blog is not a business.

The dark standout at §2 uses a three-stop gradient from #1c1832 → #2d2547 → #3a2d4d. Text on it is #faf5ed (warm cream).

Component modifier color map

Rule of thumb: affirmative = sage, caution = gold, warning = rose, neutral = lilac. New variants should follow this mapping.

Affirmative · sage.callout.good — .voice-example.yours, .compare-card.good, .niche-card.viable
Caution · gold.callout.caution — .niche-card.hard, .voice-example.bland, .grant-card.stretch
Warning · rose.callout (default) — .compare-card.bad
Neutral · lilac.path-card — .pattern-card accents
